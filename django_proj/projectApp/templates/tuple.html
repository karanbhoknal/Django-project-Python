Python Tuples


Tuple is a datatype of python used to store multiple element in single variable is called Tuple
# Creating Empty tuple
tuple=()
Print(tuple)

# Creating tuple with integer elements
tuple2=(1,2,3)
print(tuple2)


# Tuple with mixed datatype
tuple3=(101,"marri",200000.00,"HR dept")
print(tuple3)


# Creating Nested tuple
tuple2=("points",[1,2,3],(7,4,3))
print(tuple2)


# Tuple can be created without any paraenthis
# Also called tuple packing
tuple=101,"marri",200000.00,"HR dept"
print(tuple)


# Accessing Element in a tuple
tuple2=('w','R','H')
print(tuple2)
print(tuple2[1])
print(tuple2[2])


# Slicing tuple Contents
tuple2=('w','R','H','l')
print(tuple2)
print(tuple2[1:3])
print(tuple2[ :2])


# Iteration through tuple Element
tuple2=('w','R','H')
for letters in tuple2:
print("letters is",letters)

# Built in function with tuple
tuple=(23,44,88,99)
print(tuple)
print(max(tuple))
print(min(tuple))
print(sorted(tuple))
print(len(tuple))
